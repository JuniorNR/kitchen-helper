# 📁 Структура проекта

Проект **Kitchen Helper** следует архитектуре **Feature-Sliced Design (FSD)**, которая обеспечивает масштабируемость и поддерживаемость кода.

## 🏗️ Общая структура

```
kitchen-helper/
├── 📱 src/app/              # Next.js App Router (страницы)
├── 📦 src/entities/          # Бизнес-сущности
├── 🎨 src/features/          # Функциональные возможности
├── 🧩 src/widgets/           # Композитные компоненты
├── 🔧 src/shared/            # Общие утилиты и компоненты
├── ⚙️ src/configs/           # Конфигурационные файлы
├── 📄 public/                # Статические файлы
└── 📋 package.json           # Зависимости проекта
```

## 📐 Архитектурные слои

### 1. **App Layer** (`src/app/`)
Слой страниц и роутинга Next.js. См. [[📱 App|📱 App]]

### 2. **Widgets Layer** (`src/widgets/`)
Композитные компоненты, объединяющие несколько features. См. [[🧩 Widgets|🧩 Widgets]]

### 3. **Features Layer** (`src/features/`)
Бизнес-функциональность приложения. См. [[🎨 Features|🎨 Features]]

### 4. **Entities Layer** (`src/entities/`)
Бизнес-сущности и их модели данных. См. [[📦 Entities|📦 Entities]]

### 5. **Shared Layer** (`src/shared/`)
Переиспользуемые утилиты, компоненты, хуки. См. [[🔧 Shared|🔧 Shared]]

## 🔄 Принципы организации

### Импорты между слоями

- **App** может импортировать из всех слоев
- **Widgets** может импортировать из Features, Entities, Shared
- **Features** может импортировать из Entities, Shared
- **Entities** может импортировать только из Shared
- **Shared** не может импортировать из других слоев

### Структура модуля

Каждый модуль обычно содержит:
```
ModuleName/
├── index.ts                # Публичный API модуля
├── model/                  # Бизнес-логика, API, типы
└── ui/                     # UI компоненты
```

## 📝 Соглашения об именовании

- **Компоненты**: PascalCase (например, `Chat.tsx`)
- **Функции/хуки**: camelCase (например, `useChat.ts`)
- **Типы**: PascalCase (например, `ChatType.ts`)
- **Константы**: UPPER_SNAKE_CASE (например, `API_URL`)
- **Файлы**: соответствуют экспортируемому имени
