## 📄 Компонент

### `ImageSRC.tsx`

Компонент для отображения изображений с fallback.

**Функциональность**:
- Отображение изображения по URL
- Fallback на дефолтное изображение при ошибке
- Поддержка Next.js Image оптимизации
