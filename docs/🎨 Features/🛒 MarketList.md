# 🛒 MarketList Feature

Список магазинов.

## 📍 Расположение

`src/features/MarketList/`

## 📂 Структура

```
MarketList/
├── index.ts
├── model/
│   └── market.types.ts
└── ui/
    ├── MarketCard.tsx
    ├── MarketCardSkeleton.tsx
    └── MarketList.tsx
```

## 📄 Компоненты

### `MarketList.tsx`

Главный компонент списка магазинов.

**Функциональность**:
- Отображение списка магазинов
- Пагинация
- Skeleton загрузки

---

### `MarketCard.tsx`

Карточка магазина.

**Функциональность**:
- Отображение информации о магазине
- Изображения магазина
- Рейтинг
- Средняя цена
- Количество рецептов
- Продавец
- Темы магазина
- Часто покупаемые товары
- Факты о продавце
